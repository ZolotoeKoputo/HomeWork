Процессор Intel Core i5-4600 3.3GHz/5GT/s/6MB s1150

Описание Intel Core i5-4590 3.3GHz/5GT/s/6MB (BX80646I54590) s1150 BOX i5-4590 — это представитель обновленной линейки процессоров Intel Core 4-го поколения под кодовым названием микроархитектуры Haswell (Refresh).
Intel Core i5-4590 наследует все удачные технологии из прошлого, но при этом получил новое семейство чипсетов Intel 9-й серии. Обладая конструктивным исполнением разъёма LGA 1150 процессор является совместимым с нынешним парком системных плат на чипсетах Intel 8-й серии, для корректной работы которых может потребоваться обновление прошивки BIOS.
Intel Core i5-4590 выпускается по нормам 22-нм техпроцесса с использованием FinFET-транзисторов. Тактовая частота процессора составляет 3.3 ГГц, которую можно увеличить до 3.7 ГГц при задействовании режима Turbo Boost. Объем
кэш-памяти 3 уровня равен 6 МБ. Встроенная графика представлена ядром Intel HD Graphics 4600 с максимальной динамической частотой системы до 1.15 ГГц.
Пониженное энергопотребление позволит системам на базе Intel Core i5-4590 работать в режиме ожидания в 20 раз эффективнее в сравнении с системами на базе Ivy Bridge.
