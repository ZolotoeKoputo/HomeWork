Процессор Intel Core i3-7100 3.9GHz/8GT/s/3MB s1151

Описание Intel Core i3-7100 3.9GHz/8GT/s/3MB (BX80677I37100) s1151 BOX Новый процессор Intel Core i3-7100 7-го поколения, с кодовым названием микроархитектуры Kaby Lake. Предназначен для настольной платформы Intel LGA 1151.
Intel Core i3-7100 производится по стандарту 14-нм техпроцесса, имеет 2 ядра, которые работают в 4 потока на штатной тактовой частоте 3.9 ГГц. Объем кэш-памяти 3 уровня равен 3 МБ. Имеет 2-х канальный контроллер памяти DDR4 / DDR3L.
