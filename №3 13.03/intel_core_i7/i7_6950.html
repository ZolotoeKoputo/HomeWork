Процессор Intel Core i7-6950X 3.0GHz/25MB s2011-3

Описание Intel Core i7-6950X 3.0GHz/25MB (BX80671I76950X) s2011-3 BOX
Новый процессор Intel Core i7-6950X, с кодовым названием микроархитектуры Broadwell-E. Предназначен для настольной платформы Intel LGA 2011-3. Принадлежит к семейству высокопроизводительных процессоров Core i7 с большим разгонным потенциалом.
Intel Core i7-6950X производится по стандарту 14-нм техпроцесса, имеет 10 ядер, которые работают в 20 потоков со штатной тактовой частотой 3.0 ГГц, 3.5 ГГц в режиме Turbo Boost 2.0. Объем кэш-памяти 3 уровня равен 25 МБ. Имеет 4-х канальный контроллер памяти DDR4 2400/2133 и разблокированный множитель.
Intel Hyper-Threading Technology (Intel HT Technology)
Обеспечивает два потока обработки для каждого физического ядра. Многопоточные приложения могут выполнять больше задач параллельно, что значительно ускоряет выполнение работы.
Intel Turbo Boost Max 3.0
Определяет лучшую производительность ядер в процессоре и обеспечивает увеличенную производительность в ядрах с помощью возрастающей по мере необходимости частоты, пользуясь преимуществом резерва мощности и температуры.
